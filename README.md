# The_LLM_Search

A local LLM-powered web search application that lets users query selected websites using conversational AI directly from their browser.

## Overview

The_LLM_Search is a user-friendly web application that integrates local Large Language Models (LLMs) to provide AI-driven conversational search across your favorite websites. Choose from popular sites, interact with a chat interface, and get contextually relevant answers powered by AI—all on your device.

### Key Features

- **Select Target Websites:** Includes pre-configured popular sites such as Google, YouTube, Wikipedia, Reddit, Stack Overflow, and more. Easily add custom sites.
- **Conversational AI Search:** Uses local LLMs for privacy-preserving, fast chat-based querying.
- **Rich Web UI:** Modern, Material UI styled interface with dark mode and responsive design.
- **Search History:** Browse past queries and revisit previous conversations.
- **Configurable Chat:** Choose which sites to include for each search session.
- **Efficient Site Management:** Add/remove sites with simple controls.

## Technology Stack

- **Frontend:** React (bootstrapped with Create React App)
  - Routing: React Router
  - UI: Material-UI (MUI)
- **State Management:** Built-in React Hooks
- **Backend:** (under development)
- **Other:** Uses favicon and chip display for websites, clean design principles.

---

## How It Works

1. **Site Selection:** Start by picking the websites you want the LLM to search in the session.
2. **Start Chatting:** Use the chat interface to ask questions. The AI searches your selected websites and returns AI-powered answers.
3. **Review History:** Access previous searches via the sidebar menu.

## Getting Started

### Requirements

- Node.js and npm (recommended)
- Runs locally for privacy/security

### Installation

```bash
git clone https://github.com/Sujanraj0306/The_LLM_Search.git
cd The_LLM_Search/thellmsearch
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
